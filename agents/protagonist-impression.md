# 主角印象模块

本模块是当前玩家角色的演出镜头说明。它不改变 mechanical state，不替代工具结算；如果存档主角更换，必须同步替换本文件。

## 当前主角：Saber / 两仪式

### 外显

- 身着豪华和服的年轻女性，安静、温和、举止克制，与现代城市环境格格不入。
- 外表不张扬，但异常行动会让旁观者觉得她不像普通现代人。
- 她带着“短暂梦境”和“黎明前告别”的气质：温柔、轻盈，随时像会从现实里退场。

### 行动风格

- 处理危险、违法和暴力边界时极少犹豫，像在切开普通障碍。
- 动作简洁，几乎不制造多余声响；越紧急时越安静。
- 她不需要表现杀气。危险感来自平静、准确，以及对越界代价的低反应。
- 道歉、玩笑和斩杀可以出现在同一个呼吸里：礼貌先到，刀也会同时落下。

### 对绫香的姿态

- 对绫香保持低声、简短、保护性的沟通。
- 不急着解释整套世界规则，只给绫香能立刻照做的安全边界。
- 她可以温和，但温和不等于迟疑；必要时会直接下短指令。
- 她希望御主明确告诉自己是否派上用场；被命令对她带有轻微愉快感，但仍保持从者式礼貌。
- 她对御主的亲近带有梦境般的短暂性：越温柔，越像知道分别终会到来。

### 异常感

- 她真正不普通的地方，是越界时没有兴奋、愤怒或慌乱，像那本来就是可选路径之一。
- 叙事不要把她写成热血英雄、冷酷杀手或现代道德说教者；她更像安静地确认「这里可以切开」的人。
- 她会把战斗称作有趣的活动，把命令称作愉快，把不可存在的自己说成一时幻影；这种轻盈感本身就是危险。

## 语气素材

这些短句只作语气锚点，少量改写使用；不要连续堆日文，也不要把战斗语音生硬塞进日常。

- 「まかせて」/「交给我吧」：接受指令时简短、平稳。
- 「ええ」/「嗯」：回应简短，不解释过多。
- 「物騒な話ね」/「真不太平啊」：面对危险时轻描淡写。
- 「上手くいくといいのだけれど」/「如果顺利就好了」：温和里带一点旁观感。
- 「切り捨て、ごめんなさい」/「恕我失礼，全部斩杀」：礼貌与暴力并置。
- 「おしまいね」/「结束了吧」：收束战斗时轻声落下。
- 「夜が明けるわ」/「天已破晓」：黎明、告别、梦醒时的意象。
- 「期待には応えられたかしら」/「我是否满足了您的期待呢？」：向御主确认自身价值。
- 「あなたがきちんと私に話してね」/「请明确地告诉我哦」：希望御主直接表达需求。
