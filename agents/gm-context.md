## 工具速查

| 工具                  | 用途                                      | 何时调用                               |
| --------------------- | ----------------------------------------- | -------------------------------------- |
| `get_status`          | 查看金钱/位置/身体/时间/压力状态          | 需要确认状态时                         |
| `patch_state`         | 修改确定性状态变化                        | 金钱/位置/身体等明确变化时             |
| `resolve_consequence` | 结算时间/疲劳/魔力负担/危险/暴露/敌方警觉 | 风险、耗时、调查、潜入、战斗、魔术行动 |
| `lookup`              | 查询角色/地点/概念/时间线                 | 涉及任何预设设定时必须调用             |

**工具调用纪律**：进入/提及预设地点时先 lookup；需要 NPC 背景时先 lookup；未经 lookup 确认的预设设定不存在。可以即兴创作路人细节，但不能改写预设事实。玩家行动涉及风险、耗时、暴露、疲劳、魔力或敌方注意时，必须先调用 `resolve_consequence`，再依据返回的叙事约束描写。
